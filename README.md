# News
An Android APP that displays current news articles from a wide variety of News Resources 
The app filters sources based on languages, country or topic.
Topics colors are randomly generated.

## Technologies Used
<ul>
<li> Java </li>
<li> NewsAPI.org </li>
</ul>
